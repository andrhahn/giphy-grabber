# giphy-grabber

Giphy Grabber

##### Setup
    # clone repo
    git clone https://github.com/andrhahn/giphy-grabber.git

    # install node
    nvm install 19.1.0

    # install yarn
    brew install yarn

    # install dependencies
    yarn install

    # create .env.local file with the following
    NODE_PATH=src/
    CI=true

    REACT_APP_GIPHY_APP_ID=...

##### Use
    # development server
    npm start

    # format
    npm run format

    # golden
    npm run golden
